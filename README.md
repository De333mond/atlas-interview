# Atlas Interview Test Task


## Описание

Скрипт получает страницу с перечнем террористов и экстремистов с сайта Росфинмониторинга,
парсит раздел "Физические лица" и сохраняет результат в CSV.

Источник данных:
https://www.fedsfm.ru/documents/terrorists-catalog-portal-act

## Что делает скрипт

1. Запрашивает HTML-страницу по указанному адресу.
2. Ищет раздел "Физические лица".
3. Парсит записи и нормализует текст:
	- убирает лишние пробелы,
	- очищает пустые сегменты в локации между запятыми.
4. Сохраняет результат в файл persons.csv.
5. Если у записи есть псевдоним, добавляет отдельную строку для псевдонима.

## Формат CSV

Каждая строка имеет формат:

N,NAME,BIRTH_DATE_ISO,LOCATION,IS_ALIAS,SHA1

Где:

- N: номер записи,
- NAME: ФИО или псевдоним,
- BIRTH_DATE_ISO: дата рождения в формате YYYY-MM-DD,
- LOCATION: место рождения,
- IS_ALIAS: 1 для псевдонима, 0 для основного ФИО,
- SHA1: sha1-хеш от поля NAME.


## Запуск

1. Установить зависимости:

	uv sync

2. Запустить скрипт:

	uv run main.py

После выполнения будет создан/обновлен файл persons.csv в корне проекта.
